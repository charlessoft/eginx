egnix
=====

static nginx.

Install
-------

    $ git clone https://github.com/uukuguy/eginx.git
    $ cd eginx
    $ git submodule init
    $ git submodule update
    $ cd modules
    $ cd flvplay
    $ git pull origin master
    $ git checkout master
    $ cd ../rtmp
    $ git pull origin master
    $ git checkout master
    $ cd ../../
